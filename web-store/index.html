How to Install an Extension

Part 1: Prepare the Folder

  1. Download the extension's .zip file using the link on this page.
  2. Unzip the file. This will create a folder with the extension's files.
  3. Move this folder somewhere permanent and safe on your computer (like your "Documents" folder). Do not delete this folder after installation.

Part 2: Install in Chrome

  1. In a new Chrome tab, go to the address: chrome://extensions
  2. In the top-right corner, turn on Developer mode.
  3. Click the Load unpacked button that appears in the top-left.
  4. In the file window that opens, select the entire folder you prepared in Part 1.

The extension is now installed and ready to use!
